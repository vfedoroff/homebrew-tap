# Homebrew Tap

Personal Homebrew tap for my macOS applications and tools.

## How to Install

To install the cask from this tap, run:

```bash
brew install --cask vfedoroff/tap/copypathas
```

Alternatively, you can tap this repository and install the cask:

```bash
brew tap vfedoroff/tap
brew install --cask copypathas
```
